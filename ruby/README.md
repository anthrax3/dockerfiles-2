# about minimum2scp/ruby image

Ruby installed image, on Debian sid amd64

## debian packages

 * ruby (ruby2.1)
 * bundler
 * rake
 * pry

## rbenv

rbenv is installed in /opt/rbenv, and some plugins are installed.

 * ruby-build
 * rbenv-default-gem
 * rbenv-gem-rehash
 * rbenv-alias

rbenv-default-gem installs some gems for each rubies:

 * bundler
 * pry
